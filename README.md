# 🌴 SAUDI EXPLORER AI — منصة الذكاء الاصطناعي الوطنية للاستكشاف والسياحة والثقافة

![Saudi Explorer AI](https://images.unsplash.com/photo-1578895210405-907db48a7111?auto=format&fit=crop&q=80&w=1200)

**منصة سياحية وطنية متكاملة فائقة التطور، مصممة تحاكي هوية وزارة السياحة السعودية والهيئة السعودية للسياحة (روح السعودية) متوافقة مع رؤية المملكة 2030.**

---

## 🌟 أبرز مراحـل المشروع والتطبيقات المدمجة

تتكون المنصة من **18 مرحلة متكاملة** تلبي تطلعات المستثمرين، السيّاح، والجهات الحكومية:

1. **Phase 1 — PRD Specification Engine**: وثيقة متطلبات المنتج الوطنية مع محاكي المتطلبات.
2. **Phase 2 — System Architecture & Data Schema**: الهندسة المعمارية للنظام وهياكل البيانات وسجل تدفق البيانات.
3. **Phase 3 — Visual Identity & Brand System**: الهوية البصرية الرسمية، دراسة الألوان (الزمردي، الرملي، الذهب السعودي) ودليل الطباعة.
4. **Phase 4 — High-Fidelity UX Wireframes**: المخططات الهيكلية لتجربة المستخدم وسيناريوهات الرحلة.
5. **Phase 5 — Unified Auth & Nafath Integration**: نظام تسجيل الدخول الموحد، نفاذ، والتحقق البيومتري.
6. **Phase 6 — Tourist Experience Portal**: بوابات الاستكشاف، المسارات الثقافية، وتصفيات الوجهات.
7. **Phase 7 — Gemini AI Itinerary Generator**: مولّد الخطة الذكية للرحلة القائم على نماذج Gemini AI.
8. **Phase 8 — Google Maps & AR Cultural Guide**: الخرائط التفاعلية والواقع المعزز والمعالم ثلاثية الأبعاد.
9. **Phase 9 — Booking Engine**: محرك الحجوزات الموحد للفنادق، الفعاليات، والطيران.
10. **Phase 10 — Digital Wallet & Tourism Passport**: المحفظة الرقمية، جواز السفر السياحي الذكي، والتذاكر بـ QR.
11. **Phase 11 — Investor Portal**: بوابة المستثمرين للفرص السياحية والتراخيص والدراسات الجدوى.
12. **Phase 12 — Business & Merchant Portal**: بوابة أصحاب الأعمال والأنشطة السياحية.
13. **Phase 13 — Tour Guide Portal**: بوابة المرشدين السياحيين والترخيص والمسارات المعتمدة.
14. **Phase 14 — Ministry Dashboard & Vision 2030 KPIs**: لوحة قيادة وزارة السياحة ومؤشرات الأداء الوطنية.
15. **Phase 15 — Super Admin Control Center**: مركز التحكم الشامل بالمنصة والأمان وإدارة النماذج.
16. **Phase 16 — Live Interactive Demo Showcase**: العرض التفاعلي الكامل لجلسات العرض أمام المستثمرين.
17. **Phase 17 — Voice AI Assistant & 3D Avatar**: المساعد الصوتي المتكلم باللغة العربية والإنجليزية، والمرشد ثلاثي الأبعاد.
18. **Phase 18 — Digital Twin Ops & Predictive Analytics**: التوأم الرقمي للوجهات والسيموليشن وإدارة الجماهير التنبؤية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **Front-end Framework**: React 18 (Vite + TypeScript)
* **Styling & Design System**: Tailwind CSS v4 + @import "tailwindcss"
* **UI Icons**: Lucide React
* **Data Visualization**: Recharts (Charts & Gauges)
* **Audio & Speech**: HTML5 Web Speech API & Synthesizer
* **Backend Server**: Node.js + Express (Full-stack server setup)
* **AI Engine**: @google/genai (Gemini API Integration)

## 🚀 كيفية التشغيل والتطوير (Getting Started)

### 1. تثبيت الحزم (Installation)
```bash
npm install
```

### 2. تشغيل بيئة التطوير (Development Server)
```bash
npm run dev
```

### 3. بناء وإطلاق النسخة الإنتاجية (Production Build)
```bash
npm run build
npm start
```
